# Release Assets

Prepare these assets for each public release:

- one square addon/listing icon
- one wide screenshot of the reusable chat panel in the demo scene
- one screenshot showing the connected/authenticated header state
- one screenshot showing structured output or partial streaming in the transcript

Recommended source scene:

- `demo/phase7_demo.tscn`

Keep the screenshots focused on the shipped addon UI rather than editor chrome whenever possible.
