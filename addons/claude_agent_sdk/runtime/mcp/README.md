## Runtime MCP Helpers

This folder contains the scene-free, Godot-native SDK MCP tool/server helpers.

- `ClaudeMcp` provides builder helpers for tools, annotations, and JSON Schema fragments.
- `ClaudeMcpTool` and `ClaudeMcpToolAnnotations` model in-process MCP tool definitions.
- `ClaudeSdkMcpServer` stores precomputed tool metadata and handlers for the runtime JSON-RPC bridge.
