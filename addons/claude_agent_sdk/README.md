# Claude Agent SDK Addon

This directory is the future distributable addon payload.

The addon now contains a working runtime core plus the first Godot integration layer.

Current subtrees:

- `runtime/`
- `runtime/transport/`
- `runtime/protocol/`
- `runtime/messages/`
- `runtime/parser/`
- `runtime/adapters/`
- `ui/`
- `icons/`

Development-only tooling such as tests, probes, and GdUnit4 should stay outside this addon payload boundary.
