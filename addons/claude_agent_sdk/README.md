# Claude Agent SDK Addon

This directory is the future distributable addon payload.

Phase 3 scaffolds the runtime layout only. The implementation is intentionally still thin.

Current subtrees:

- `runtime/`
- `runtime/transport/`
- `runtime/protocol/`
- `runtime/messages/`
- `runtime/parser/`
- `runtime/adapters/`
- `ui/`
- `icons/`

Development-only tooling such as tests, probes, and GdUnit4 should stay outside this addon payload boundary.
